# 🚀 Cursor 啟動指南

> **每次啟動 Cursor 時請先閱讀此文件**

## 📋 快速檢查清單

### 1. 運行開發環境檢查
```bash
pnpm dev:check
```

### 2. 啟動開發服務器（如果未運行）
```bash
pnpm dev --port 4321
```

### 3. 閱讀開發規範
- [DEVELOPMENT-GUIDE.md](./DEVELOPMENT-GUIDE.md) - 快速開發指南
- [docs/i18n-GUIDE.md](./docs/i18n-GUIDE.md) - 完整 i18n 規範

## 🚨 重要提醒

### 新功能開發流程
1. **先寫** `src/i18n/messages/zh-TW/模組.json`
2. **再寫** 頁面代碼使用 `{messages.模組?.鍵名}`
3. **檢查** `pnpm scan:i18n`
4. **同步** `pnpm i18n:sync`

### 絕對禁止
- ❌ 硬編碼中文或英文
- ❌ 先寫頁面代碼再抽取文案
- ❌ 忘記同步到其他語言

### 每次提交前
```bash
pnpm scan:i18n     # 檢查硬編碼
pnpm i18n:sync     # 同步到所有語言
```
