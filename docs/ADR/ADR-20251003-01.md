We adopt Astro SSG with 34 locales (incl. RTL), SSR metrics fallbacks for SEO, and a single React island for live metric updates (every 60s). GitHub Pages deploy uses gh-pages or /docs without Actions.

Consequences:
- Minimal JS shipped per page (island only where used)
- Strong SEO guarantees with pre-rendered content
- BASE_URL-aware links for GitHub Pages subpath
