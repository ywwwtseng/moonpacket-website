## 📋 使用流程

### 下次啟動 Cursor 時：

1. **閱讀啟動指南**
   ```bash
   cat .cursor-startup.md
   ```

2. **檢查開發環境**
   ```bash
   pnpm dev:check
   ```

3. **閱讀開發規範**
   ```bash
   cat DEVELOPMENT-GUIDE.md
   ```

4. **開始開發新功能**
   - 先寫 `src/i18n/messages/zh-TW/模組.json`
   - 再寫頁面代碼使用 `{messages.模組?.鍵名}`
   - 運行 `pnpm scan:i18n` 檢查
   - 運行 `pnpm i18n:sync` 同步
